Basic search

Here you can perform a search utilising HDAS advanced options, and build a search strategy.

 

What’s new?

 

• The database selection now takes place on this (or the basic search) screen. This approach gives the following benefits:

 

     - A simplified approach, with less clicks required before starting the search.

     - The possibility to select multiple other databases without going to a different screen (currently not possible, as the Select another’ database drop-down only allows the user to select one database).

 

• You can now perform an advanced search when not logged in (on PubMed only), which will allow users who have forgotten their login details to start searching whilst waiting to recover their username or password.

 

• You can move onto the basic search screen at any time.

 

• You can now see search results without clicking:

 

     - If you select ‘Advanced search results’ under ‘What do you want to see?’, then your search results are displayed as soon as the search completes.

     - If you select ‘ Search history only’, then you can build up a search strategy before you choose to see the results.

 

 

**Don’t forget*

This is an interactive design, not actual working software. It does not perform real searches.

This is a work in progress: some of the functionality isn’t included yet, but we’re not forgetting about it.
