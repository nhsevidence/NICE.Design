What’s new?

 

Improvements made in response to your feedback

 

•  The default number of results shown per page has increased. The default is currently 25, but your feedback is welcome as to whether it needs to be higher.

•  We’ve maximised the space available for showing results by moving the ‘Share or Save’ options into a pop-up.

•  Abstracts are now visible by default in the search results.

•  The current clipboard functionality is still available but has been renamed as ‘Save results’. The label is likely to change again.

•  You can edit a search. (To do so, expand the ‘Search history’ and click ‘Edit’)

 

What’s different from advanced search results?

 

•  The ‘Search history’ is collapsed by default.

•  When editing a search in the search history, you only get basic options.

 

Additional functionality

 

Functionality that’s not yet included in this design, or not obvious from it, includes:

•  When saving results (currently called ‘Clipboard’), the intention is to allow you to save results across sessions.

•  A spellchecker will be integrated with the search box.

•  Full text links will be available from the search results view, as is currently the case

•  Clicking on an article’s title will lead to a detailed view.

•  Users will be able to obtain a list of articles for which the full text is available, and a list of articles for which it isn’t.

 

(Note that this is not an exhaustive list!)

 

 

**Don’t forget*

This is an interactive design, not actual working software. It does not perform real searches.

This is a work in progress: some of the functionality isn’t included yet, but we’re not forgetting about it.
