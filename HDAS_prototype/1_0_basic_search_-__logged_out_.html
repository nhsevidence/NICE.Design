Here you can perform a keyword search and view search results. This replaces the current ‘Simple search’ functionality. Basically, the concept of ‘Simple search’ has been extended so that users can perform a basic search equivalent to that currently offered by ‘Simple search’ on any database (rather than just PubMed).

 

What’s new?

 

•  The database selection now takes place on this (or the advanced search) screen. This approach delivers the following benefits:

 

       - A simplified approach, with fewer clicks required before starting the search.

       - The possibility to select multiple other databases without going to a different screen

          (currently not possible, as the ‘Select another’ database drop-down only allows the user to select one database).

 

•  You can now perform a basic search on any of the databases served by HDAS (provided you’re logged in), which will allow non information specialists to search without being unsettled by the complexity and multitude of options offered by an advanced search.

 

•  You can move onto the advanced search screen at any time.

 

**Don’t forget*

This is an interactive design, not actual working software. It does not perform real searches.

This is a work in progress: some of the functionality isn’t included yet, but we’re not forgetting about it.
